# Codex List Games

A small Vue 3 + Vite game project. The current version keeps only the Snake game.

## Features

- Snake game rendered on canvas
- Keyboard controls
- Touch swipe controls
- Score and best-score tracking with `localStorage`
- Responsive layout for desktop and mobile

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
